# Rahul Chikkala's Portfolio

This is a Jekyll-powered portfolio website using the Hacker theme.

## Local Development

To run this locally:

1. Install Ruby and Jekyll:
   ```bash
   gem install jekyll bundler
   ```

2. Navigate to your project directory and run:
   ```bash
   bundle exec jekyll serve
   ```

3. Open `http://localhost:4000` in your browser

## Deployment to GitHub Pages

1. Push this repository to GitHub as `your-username.github.io`
2. GitHub Pages will automatically build and deploy
3. Your site will be live at `https://your-username.github.io`

## Theme

This portfolio uses the **Hacker** Jekyll theme: https://github.com/pages-themes/hacker

## File Structure

```
your-username.github.io/
├── _config.yml          # Jekyll configuration
├── index.md             # Homepage content
├── README.md            # Repository README
└── _layouts/            # (Optional) Custom layouts
```

## Customization

Edit `index.md` to update your content. The Hacker theme automatically styles your markdown.

To change the theme, update the `remote_theme` in `_config.yml`.
